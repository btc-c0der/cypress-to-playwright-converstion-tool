## 🚀 Quick Start Guide

### 1. Setup (One Command)
```bash
./setup.sh
```

### 2. Start Portal
```bash
source venv/bin/activate
python main.py
```

### 3. Access Portal
Visit: http://localhost:7860

### 4. Optional: Configure AI
Edit `.env` file with your Hugging Face token for full AI features.
